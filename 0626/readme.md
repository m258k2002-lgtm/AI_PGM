## 🐍 파이썬 리스트(List) 완벽 마스터 가이드

파이썬을 배우면서 가장 먼저 만나게 되고, 또 가장 많이 사용하게 되는 핵심 데이터 구조가 바로 리스트(List)입니다. 리스트는 여러 개의 데이터를 하나의 변수에 차곡차곡 담아둘 수 있는 '다용도 보관함' 혹은 '기차'와 같습니다.

리스트를 자유자재로 다루기 위해 꼭 알아야 할 핵심 개념들을 단계별로 살펴보겠습니다.

### 1. 리스트의 3가지 핵심 특징

* **순서가 있습니다 (Ordered):** 리스트 안에 들어간 데이터들은 무작위로 섞이지 않고, 들어간 순서대로 자기만의 '번호표(인덱스)'를 부여받습니다. 이 번호표를 통해 언제든 원하는 데이터를 정확히 짚어낼 수 있습니다.
* **자유롭게 수정할 수 있습니다 (Mutable):** 한 번 만들어진 리스트라도 굳어진 것이 아닙니다. 언제든지 새로운 데이터를 뒤에 추가하거나, 중간에 끼워 넣거나, 필요 없는 데이터를 삭제할 수 있습니다.
* **아무거나 다 담을 수 있습니다 (Heterogeneous):** 숫자면 숫자, 문자면 문자만 담아야 하는 다른 프로그래밍 언어들과 달리, 파이썬 리스트는 몹시 관대합니다. `[1, "사과", 3.14, True]`처럼 숫자, 문자, 소수점, 논리값 심지어 또 다른 리스트까지 한 바구니에 마구 섞어서 담아도 전혀 문제가 없습니다.

---

### 2. 데이터 꺼내기: 인덱싱(Indexing)과 슬라이싱(Slicing)

리스트 안에 담긴 데이터를 꺼내어 쓰려면 인덱스(Index)라는 번호표 체계를 이해해야 합니다.

**📍 인덱싱 (하나씩 꺼내기)**

* **정방향:** 파이썬은 숫자를 **0부터** 셉니다. 첫 번째 칸은 0번, 두 번째 칸은 1번입니다.
* **역방향:** 뒤에서부터 셀 때는 **-1부터** 시작합니다. 리스트가 아무리 길어도 제일 마지막 데이터는 항상 `-1번`입니다.

```python
fruits = ["사과", "바나나", "체리", "망고"]

print(fruits[0])   # 출력: 사과 (첫 번째)
print(fruits[-1])  # 출력: 망고 (마지막)

```

**📍 슬라이싱 (원하는 만큼 잘라내기)**
리스트의 일부분만 통째로 잘라내어 새로운 리스트를 만들고 싶을 때는 `[시작_인덱스 : 끝_인덱스]` 방식을 사용합니다. (단, 끝 인덱스의 바로 앞부분까지만 잘라냅니다.)

```python
numbers = [10, 20, 30, 40, 50]

print(numbers[1:4]) # 출력: [20, 30, 40] (1번부터 3번까지만 추출)

```

---

### 3. 데이터 조작하기: 추가와 삭제

리스트를 내 마음대로 주무르는 주요 명령어(메서드)들입니다.

**➕ 데이터 추가하기**

* **`.append(값)`:** 리스트의 가장 맨 끝에 데이터를 하나 덧붙입니다. 가장 많이 쓰입니다.
* **`.insert(인덱스, 값)`:** 내가 지정한 특정 위치(인덱스)에 데이터를 새치기하듯 끼워 넣습니다.

```python
cart = ["우유", "계란"]
cart.append("식빵")       # ["우유", "계란", "식빵"]
cart.insert(0, "버터")    # ["버터", "우유", "계란", "식빵"] (0번 자리에 끼워넣기)

```

**➖ 데이터 삭제하기**

* **`.remove(값)`:** 인덱스가 아닌, '값' 자체를 찾아서 지웁니다. (동일한 값이 여러 개면 가장 첫 번째 것만 지웁니다.)
* **`.pop(인덱스)`:** 지정한 위치(인덱스)의 데이터를 삭제하면서, 동시에 그 값을 바깥으로 꺼내어 줍니다.

```python
colors = ["빨강", "파랑", "노랑"]
colors.remove("파랑")     # ["빨강", "노랑"] ("파랑"이라는 값을 찾아 삭제)

```

---

### 4. 알아두면 든든한 필수 내장 함수

파이썬에는 리스트를 쉽게 다룰 수 있도록 도와주는 마법 같은 함수들이 내장되어 있습니다. 이 네 가지만 알아도 웬만한 계산은 다 할 수 있습니다.

* `len(리스트)`: 리스트 안에 총 **몇 개**의 데이터가 들어있는지 길이를 알려줍니다.
* `sum(리스트)`: (숫자 리스트일 경우) 안에 있는 모든 숫자의 **합계**를 구해줍니다.
* `max(리스트)`: 리스트 안에서 가장 **큰 값**을 찾아줍니다.
* `min(리스트)`: 리스트 안에서 가장 **작은 값**을 찾아줍니다.

```python
scores = [85, 90, 75, 100]

print(len(scores))  # 출력: 4 (총 4개)
print(sum(scores))  # 출력: 350 (모두 더한 값)
print(max(scores))  # 출력: 100 (가장 큰 수)

```

# 📦 파이썬 컨테이너 자료구조 완벽 가이드

파이썬에서 **컨테이너(Container)**란 여러 개의 데이터를 하나의 단위로 묶어서 관리할 수 있는 자료구조를 말합니다. 목적과 데이터의 특성에 따라 크게 4가지 기본 내장 컨테이너를 제공합니다.

---

## 1. 핵심 컨테이너 한눈에 비교

| 자료구조 | 선언 기호 | 순서 보장 (Ordered) | 수정 가능 여부 (Mutable) | 중복 허용 | 주요 용도 |
| :---: | :---: | :---: | :---: | :---: | :--- |
| **List (리스트)** | `[ ]` | O | 🟢 가변 (Mutable) | O | 순차적인 데이터 보관, 범용적 용도 |
| **Tuple (튜플)** | `( )` | O | 🔴 불변 (Immutable) | O | 수정되면 안 되는 고정 데이터, 다중 반환값 |
| **Dictionary (딕셔너리)** | `{Key: Value}` | O (3.7+) | 🟢 가변 (Mutable) | Key 불가, Value 가능 | 키-값 쌍의 매핑 구조, 빠른 탐색 |
| **Set (집합)** | `{ }` | X | 🟢 가변 (Mutable) | X | 중복 제거, 집합 연산(교/합/차집합) |

---

## 2. 컨테이너별 상세 분석 및 핵심 문법

### ■ List (리스트) - 가장 범용적인 가변 시퀀스
리스트는 순서가 보장되며 데이터를 자유롭게 추가, 수정, 삭제할 수 있는 가장 대표적인 자료구조입니다.

```python
# 리스트 기본 선언 및 조작
fruits = ["사과", "바나나"]
fruits.append("체리")       # 맨 뒤에 추가 -> ["사과", "바나나", "체리"]
fruits.insert(1, "망고")    # 특정 위치 삽입 -> ["사과", "망고", "바나나", "체리"]
fruits.remove("바나나")     # 값으로 삭제 -> ["사과", "망고", "체리"]

# 인덱싱과 슬라이싱
print(fruits[0])            # '사과' (첫 번째 요소)
print(fruits[-1])           # '체리' (마지막 요소)
print(fruits[0:2])          # ['사과', '망고'] (0번부터 1번까지 추출)

```

### ■ Tuple (튜플) - 안전성이 보장된 불변 시퀀스

튜플은 리스트와 거의 동일하지만, 한 번 생성하면 내부 값을 절대 변경할 수 없다(Immutable)는 결정적인 차이가 있습니다.

```python
# 튜플 기본 선언
point = (10, 20)
user_info = ("홍길동", 30, "서울")
single_element = (5,)       # 원소가 하나일 때는 반드시 쉼표(,) 필요

# 튜플 활용: 다중 반환 및 구조 분해 (Unpacking)
x, y = point
print(f"x 좌표: {x}, y 좌표: {y}") # x 좌표: 10, y 좌표: 20

# point[0] = 50  # ❌ TypeError 발생! (수정 불가)

```

> **💡 Tip (리스트 vs 튜플):** 데이터가 계속 바뀌어야 한다면 `List`를, 좌표값이나 시스템 설정값처럼 고정되어야 한다면 `Tuple`을 사용하는 것이 정석입니다.

### ■ Dictionary (딕셔너리) - 키(Key)와 값(Value)의 매핑

사전식 구조로, 단어(Key)와 뜻(Value)이 짝을 이루는 형태입니다. 데이터 양이 많아도 Key를 통해 값을 찾는 속도가 초고속(O(1))입니다.

```python
# 딕셔너리 기본 선언 및 조작
user = {
    "name": "이순신",
    "age": 45,
    "city": "부산"
}

# 데이터 접근 및 추가
print(user["name"])         # '이순신'
user["email"] = "lee@com"  # 새로운 키-값 추가
user["age"] = 46            # 기존 값 수정

# 안전하게 값 가져오기
job = user.get("job", "무직") # 'job' 키가 없으면 기본값 '무직' 반환

```

### ■ Set (집합) - 중복을 허용하지 않는 고유 데이터

순서가 없기 때문에 인덱싱(`[0]`)이 불가능하지만, **중복된 데이터를 완전히 배제**하며 교집합, 합집합 등 집합 연산에 특화되어 있습니다.

```python
# 집합 선언 및 중복 제거
numbers = {1, 2, 2, 3, 3, 3}
print(numbers)              # {1, 2, 3} (중복 자동 제거)

# 리스트의 중복 제거용으로 자주 쓰임
raw_list = ["A", "B", "A", "C", "B"]
unique_list = list(set(raw_list)) # ['A', 'B', 'C']

# 집합 연산
set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}

print(set_a & set_b)        # 교집합: {3, 4}
print(set_a | set_b)        # 합집합: {1, 2, 3, 4, 5, 6}
print(set_a - set_b)        # 차집합: {1, 2}

```

## 3. 컨테이너 활용 필수 내장 함수

모든 컨테이너 객체에 공통적 혹은 유연하게 적용되어 데이터를 다루기 쉽게 해주는 필수 함수들입니다.

```python
nums = [10, 20, 30]

# 데이터 길이 및 수학 연산
print(len(nums))  # 3 (데이터의 총 개수)
print(sum(nums))  # 60 (총합)
print(max(nums))  # 30 (최댓값)
print(min(nums))  # 10 (최솟값)

# 정렬
unordered = [4, 1, 3, 2]
ordered = sorted(unordered) # [1, 2, 3, 4] (정렬된 새로운 리스트 반환)

# 멤버십 테스트 (in 연산자)
fruits = ["apple", "banana"]
print("apple" in fruits) # True
print("grape" in fruits) # False

```
